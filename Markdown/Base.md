# Escaping Characters

Characters You Can Escape

| Character | Name  |
| ---       | ---   |
| \\        | backslash |
| \`        | backtick (see also escaping backticks in code) |
| \*        | asterisk |
| \_        | underscore |
| \{ \}     | curly braces |
| \[ \]     | brackets |
| \< \>	    | angle brackets |
| \( \)	    | parentheses |
| \#        | pound sign |
| \+	    | plus sign |
| \-        | minus sign (hyphen) |
| \.        | dot |
| \!        | exclamation mark |
| \|        | pipe (see also escaping pipe in tables) |
